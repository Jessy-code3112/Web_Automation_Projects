# Web Automation & Interaction Portfolio

A repository demonstrating browser automation, dynamic UI interactions, and automated reporting using Python and Playwright.

## Project Directory

### 1. Book Category Dynamic Navigator (`book_auto.py`)
- **Target:** An e-commerce bookstore layout.
- **Actions:** Automatically boots up a Chromium browser, targets and clicks specific sidebar categories (`Travel`), captures a localized dashboard snapshot, and parses active item elements.
- **Outputs:** Generates `travel_books_dashboard.png` and structure-maps contents directly into `travel_books_report.csv`.

## Tech Stack
- Python 3
- Playwright (Chromium Automation Engine)
- Pandas (Data Structuring)
